# Data Analysis and Visualization using Python

This repository contains Python notebooks created using Google Colab for learning and practicing data analysis, visualization, and numerical computing.

## Contents

### 1. NumPy (`numpy.ipynb`)

* Array creation and manipulation
* Mathematical operations
* Statistical functions
* Indexing and slicing

### 2. Pandas (`pandas.ipynb`)

* DataFrame creation
* Data cleaning and preprocessing
* Data selection and filtering
* Data analysis operations

### 3. Line Plot (`line_plot.ipynb`)

* Creating line charts using Matplotlib
* Customizing colors, markers, and styles
* Adding labels, titles, and legends

### 4. Pie Chart (`pie_chart.ipynb`)

* Creating pie charts
* Using labels and percentages
* Exploding pie slices
* Customizing chart appearance

### 5. NBA Data Analysis (`nba_file.ipynb`)

* Working with NBA datasets
* Data exploration and visualization
* Statistical analysis of player and team data
